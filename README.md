


# Doctor Sahab – AI Healthcare Assistant

Doctor Sahab is an AI-powered healthcare assistant designed to provide quick medical support in both emergency and everyday situations. It allows users to interact with a chatbot, locate nearby medical facilities, and take immediate action when needed.


## Features

* AI chatbot for basic healthcare guidance and user queries
* Nearest medical facility locator
* Emergency assistance through quick ambulance calling
* Simple and user-friendly web interface


## Tech Stack

* Frontend: HTML
* Backend: Python
* Database: SQLite
* Concepts: Basic AI logic, backend integration


## Project Structure

Doctor-Sahab/
─ backend_simple.py
─ doctor_sahab_standalone.py
─ start_backend.py
─ doctor_sahab_frontend.html
─ doctor_sahab.db
─ .env (example)


## How to Run

1. Clone the repository

git clone https://github.com/harryexe69/Doctor-Sahab.git
cd Doctor-Sahab

2. Install dependencies

pip install -r requirements.txt

3. Run the backend

python start_backend.py

4. Open the frontend by running the HTML file in a browser
